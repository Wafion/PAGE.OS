# PageOS

A futuristic sci-fi themed e-book reader and library app built with Next.js. Features terminal-inspired UI, book discovery from Project Gutenberg, distraction-free reading modes, bookmark system, and Firebase sync.

## Tech Stack
- Next.js 14, React 18, TypeScript
- Radix UI, Tailwind CSS, Framer Motion
- Firebase, PDF.js

## Quick Start
1. `npm install`
2. `npm run dev`
3. Visit http://localhost:3000
